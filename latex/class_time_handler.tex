\doxysubsection{Detailed Description}
The \doxylink{class_time_handler}{Time\+Handler} class provides various methods for time operations. 

It includes functionalities such as retrieving local time, extracting the date from a time string in the format dd.\+mm.\+yyyy hh\+:mm\+:ss, obtaining the date with the month represented as a name, retrieving the UTC time of a local time, and getting the current date.

{\footnotesize\ttfamily std\+::string Time\+Handler\+::get\+UTC (\begin{DoxyParamCaption}\item[{const std\+::string \&}]{input }\end{DoxyParamCaption})}



Converts a local time in the format DD.\+MM.\+YYYY HH\+:\+MM\+:SS to UTC time. 

This function takes a local time string as input, checks if it\textquotesingle{}s not empty, and then extracts and converts the date and time components to UTC. The input string is first trimmed on the right. Then, the year, day, and month are extracted from the date part of the string. The hour, minute, and second are extracted from the time part of the string. A local time is created with these values, and a time zone for Germany (CET+01\+:00\+:00) is applied to it. The local time is then converted to UTC and returned as a string. If any part of the date or time is invalid, an error message is shown with exc() and the function returns an empty string.
