std::string UtilityMethods::removeExtraSpaces ( std::string  s)
static

Removes extra spaces from the given input string.

This method eliminates extra spaces within the input string. It loops through the characters and skips whitespace characters according to certain conditions. At the end only one space is permitted and left within the string. If we have LH 2309 and call this method on this string, it returns LH 2309, so one space less.

Parameters
sThe input string containing spaces to be removed.
Returns
The string with extra spaces removed.
